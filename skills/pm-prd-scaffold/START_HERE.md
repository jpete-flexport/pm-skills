# Start Here - PM PRD Scaffold Package

Welcome! This package contains a Claude Code skill that helps Product Managers generate structured PRDs quickly.

---

## For Users (Installing the Skill)

**Start with:** `INSTALLATION.md`

Quick install:
```bash
cp -r pm-prd-scaffold ~/.claude/skills/
# Restart Claude Code
/pm:prd-scaffold
```

Then read `README.md` for full documentation.

## One-Minute Overview

This skill:
1. Asks you ~10 questions about your project
2. Generates a complete PRD in your company's format
3. Includes success metrics, personas, requirements, and ACs
4. Saves you 2-4 hours of writing time
5. Keeps your PRDs consistent and complete

**Time investment:** 20-30 minutes
**Output:** Complete PRD draft (70% done, you refine the rest)

**Try it:** `/pm:prd-scaffold`
